# 郑轻大健康打卡脚本docker仓库

此仓库可以帮助你快速构建打卡脚本所需环境🎁

你可以用一下几条命令快速开始：

```shell
git clone https://github.com/xwwwb/ZZULI-healthreport-docker.git
cd ZZULI-healthreport-docker
vi run.py # 修改配置文件
docker build -t zzulidaka:1.0 .
docker run --rm zzulidaka:1.0 # 运行一次，可以将此命令添加进crontab
```
